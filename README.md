
# Toll Tracker

A web application where transportation companies and drivers can pay tolls en route prior to their journey for an efficient journey & automate the entire system.

## The System has these following functionalities :

- Full Frontend System with React.js.
- Backend with Node.js, Express.js and MySQL for database.
- Users Authentication and Authorization with jwt.
- Data validation with YUP.
- Integrated payment gateway with - Stripe API.
- Generate PDF invoice with barcode after successful payment completion.
- Scan barcode and verify successful payment.
